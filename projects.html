Innovation

Here are some of the projects I have worked on:

  • Create a mobile app for a local restaurant
  • Create an e-commerce website for a coffee shop
  • Tutor k-12 students

Create a mobile app for a local restaurant

I used python Tkinter to create a Graphical User Interface (GUI) prototype so customers can order food via that GUI application. The system provides Add, Delete, and Edit functions, in addition to the Exit function. The system includes two classes, customers and orders.

Create an e-commerce website for a coffee shop

I used html, css, and javascript to create the website, the front-end. I used node.js, javascript, mysql, jquery, and Ajax to create the back-end of the website. The website displays all products and services the coffee shop offers. Customers can make orders online. There is a Contact form where customers can provide feedback.
